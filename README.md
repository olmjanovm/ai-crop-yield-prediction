# ai-crop-yield-prediction
# AI Crop Yield Prediction Project

This project was created for Pearson BTEC Unit 21: Introduction to Artificial Intelligence.

## Project topic
Predicting agricultural crop yield based on weather data and pesticide usage.

## Files included
- df_yield_merged.csv — dataset used in the project
- KNIME workflow file — AI model built in KNIME
- Report document — full assignment documentation

## Tools used
- KNIME Analytics Platform
- Regression model
- Weather and agricultural dataset
